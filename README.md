# buncare-romania

Sursa: [IGSU Romania](https://www.igsu.ro/Resources/Documente_Generale/Situatia_adaposturilor_de_protectie_civila.pdf)

## Structura

Ambele fisiere .csv si .json contin aceleasi date, oferite in cele doua formate:

- list.csv
- list.json
